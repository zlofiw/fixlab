# FixLab

Учебный веб-сайт сервисного центра электроники на React + Vite + Tailwind CSS v4.

## Что реализовано

- Главная страница с обзором бизнес-процесса
- Каталог услуг и калькулятор стоимости
- Форма создания заявки
- Трекинг статуса ремонта по номеру заявки и коду доступа
- Локальная бизнес-логика расчета SLA, очереди, стоимости и скидок
- Локализация под Казахстан: `ru-KZ`, валюта `KZT`

## Технологии

- React 19
- Vite 7
- TypeScript
- Tailwind CSS v4
- React Router

## Запуск

```bash
npm install
npm run dev
```

## Проверка качества

```bash
npm run lint
npm run build
```

## Примечание

Данные заявок хранятся в `localStorage` браузера (ключ `fixlab.tickets.v2`).
